Services

Chatbot

Je développe des chatbots de plusieurs types pour les entreprises et les particuliers. J'ai une certaine expérience dans le développement de chatbot pour communiquer avec les états financiers.

Moddèles de classification

Les modèles de classification nous permettent de classer en fonction de caractéristiques individuelles. Il est utilisé pour prédire les maladies en fonction des symptômes, des e-mails (spam ou non), du caractère rentable ou non d'un investissement, etc.

Modèles de régression

Les modèles de régression permettent de prédire une donnée spécifique. Supposons que vous ayez vos revenus des 10 dernières années, nous pouvons créer un modèle pour prédire votre revenu annuel sur 5 ans. Les entreprises utilisent ces modèles pour prédire les éléments des états financiers. Ces modèles sont utilisés pour prédire la démographie et bien d’autres encore.

Computer vision

La vision par ordinateur est une méthode d’IA permettant de reconnaître ou de classer des images. J'ai juste quelques expériences dans ce domaine de l'IA.

Plateforme web

L'un des domaines informatiques dans lesquels j'ai le plus d'expérience est le développement Web statique à dynamique. Je développe des applications web de toutes sortes.

Application mobile

En plus du développement Web, je développe également des applications mobiles. Qu'il soit natif Android ou IOS via un framework cross platforme, j'ai une certaine expérience dans ce domaine.
